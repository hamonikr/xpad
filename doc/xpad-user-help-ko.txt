<span size="large"><b>Xpad 스티커 메모</b></span>

바탕화면에 간편한 메모를 붙여 사용할 수 있습니다. 모든 메모는 자동으로 저장됩니다.

<span size="medium"><b>🎯 기본 사용법</b></span>

• <b>새 메모</b>: <tt>Ctrl+N</tt> 또는 트레이 아이콘 클릭
• <b>메모 이동</b>: <tt>Ctrl</tt> + 마우스 드래그
• <b>크기 조정</b>: 우하단 모서리 드래그
• <b>메모 닫기</b>: <tt>Ctrl+W</tt> 또는 <tt>Shift+Del</tt>

<span size="medium"><b>⚙️ 주요 설정</b></span>

<u>환경설정 열기</u>
• 메모에서 우클릭 → 메모 → 편집 → 환경설정
• 트레이 아이콘 우클릭 → 환경설정

<u>개별 메모 설정</u>
• 메모에서 우클릭 → 메모 → 속성 (색상, 글꼴 변경)
• 메모에서 우클릭 → 메모 → 보기 (도구모음, 창장식 등)

<u>데이터 저장 위치 변경</u>
• 환경설정 → 데이터 디렉토리
• 클라우드 저장소 연동 가능

<span size="medium"><b>🎨 텍스트 꾸미기</b></span>

텍스트를 선택한 후 우클릭하면 다음 스타일을 적용할 수 있습니다:
• <b>굵게</b> (<tt>Ctrl+B</tt>)  • <i>기울임</i> (<tt>Ctrl+I</tt>)  • <u>밑줄</u> (<tt>Ctrl+U</tt>)

<span size="medium"><b>⌨️ 유용한 단축키</b></span>

<tt>F1: 도움말        Ctrl+A: 모두 선택
Ctrl+Q: 종료         Ctrl+Z: 실행 취소
Ctrl+N: 새 메모      Ctrl+Y: 다시 실행</tt>

<span size="medium"><b>💡 팁</b></span>

• 트레이 아이콘을 활성화하면 더 편리하게 사용할 수 있습니다
• 메모가 많을 때는 "모두 보이기/숨기기" 기능을 활용하세요
• 데이터 디렉토리를 클라우드 폴더로 설정하면 여러 컴퓨터에서 동기화됩니다
