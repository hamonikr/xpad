<span size="large"><b>Xpad Sticky Notes</b></span>

Create convenient sticky notes on your desktop. All notes are automatically saved.

<span size="medium"><b>🎯 Basic Usage</b></span>

• <b>New Note</b>: <tt>Ctrl+N</tt> or click tray icon
• <b>Move Note</b>: <tt>Ctrl</tt> + mouse drag
• <b>Resize Note</b>: Drag bottom-right corner
• <b>Close Note</b>: <tt>Ctrl+W</tt> or <tt>Shift+Del</tt>

<span size="medium"><b>⚙️ Main Settings</b></span>

<u>Open Preferences</u>
• Right-click note → Pad → Edit → Preferences
• Right-click tray icon → Preferences

<u>Individual Note Settings</u>
• Right-click note → Pad → Properties (change colors, fonts)
• Right-click note → Pad → View (toolbar, window decorations, etc.)

<u>Change Data Storage Location</u>
• Preferences → Data Directory
• Can sync with cloud storage

<span size="medium"><b>🎨 Text Styling</b></span>

Select text and right-click to apply these styles:
• <b>Bold</b> (<tt>Ctrl+B</tt>)  • <i>Italic</i> (<tt>Ctrl+I</tt>)  • <u>Underline</u> (<tt>Ctrl+U</tt>)

<span size="medium"><b>⌨️ Useful Shortcuts</b></span>

<tt>F1: Help           Ctrl+A: Select All
Ctrl+Q: Quit          Ctrl+Z: Undo
Ctrl+N: New Note      Ctrl+Y: Redo</tt>

<span size="medium"><b>💡 Tips</b></span>

• Enable the tray icon for easier access
• Use "Show All/Hide All" when you have many notes
• Set data directory to a cloud folder for multi-computer sync
